# Metadata API

> A RESTful API to get metadata from web pages.

## Documentation

Full documentation is available in the [`docs/`](./docs/README.md) directory:

## Quick Start

Request metadata:

```bash
curl https://metadata-api.carllos-nc.workers.dev/metadata?url=https://hono.dev/
```
